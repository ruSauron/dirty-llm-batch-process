Скрипт, который в лоб целиком скармливает API OpenAI все текстовые файлы в папке по 1 и сохраняет результат в одноимённые текстовые файлы в другой папке.

Планируется выгружать субтитры целого плейлиста и потом массово их пересказывать покороче. Сейчас субтитры сохраняю вручную через userscript

Поменяйте SYSTEM_PROMPT на желаемый перед запуском!

Возможно, добавлю поиск шаблона в папке скрипта и нарезку текста на n символов, но сейчас скрипт справляется с задачей, а для перевода по частям есть гораздо лучшие инструменты
